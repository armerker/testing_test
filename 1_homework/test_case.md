| ID | Название | Шаги | Ожидаемый результат |

| TC-18 | productId = 0 | 1. Отправить GET запрос с валидным токеном на /products/0/status | 404 Not Found (или 400 ) |
| TC-19 | productId = -1 | 1. Отправить GET запрос с валидным токеном на /products/-1/status | 404 Not Found (или 400 ) |
| TC-20 | productId = abc (не число) | 1. Отправить GET запрос с валидным токеном на /products/abc/status | 404 Not Found |
| TC-21 | Неподдерживаемый метод POST | 1. Отправить POST запрос с валидным токеном на /products/123/status | 405  |
